                # Mechanism

                Vue Marquee makes use of a visual illusion that achieves the scrolling effect. The inner element moves along the x-axis (for horizontal marquees) or the y-axis (for vertical marquees) and get restarted once it reaches second half.

                It involves no DOM manipulation. CSS transform composition layer is engaged efficient GPU rendering.

                # Asynchronous Images

                View Code 
                <template>
    <client-only>
        <marquee class="marquee-demo-async"
                 duration-per-slide="1000"
                 direction="rtl">
            <div class="slide"
                 v-for="{ id, author, } in asyncData"
                 v-bind:key="id">
                <img v-bind:alt="author"
                     v-bind:src="`https://picsum.photos/id/${ id }/400/300`">
            </div>
        </marquee>
    </client-only>
</template>

<script>
export default {
    name: "MarqueeDemoAsync",
    data () {
        return {
            asyncData: [],
        };
    },
    async mounted () {
        const response = await fetch("https://picsum.photos/v2/list?page=2&limit=5");
        try {
            this.asyncData = await response.json();
        } catch (e) {
            console.error(e);
        }
    }
}
</script>
